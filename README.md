## Development
### Dependencies

- nodejs ( [npm][rbenv])
- webpack
- create-react-app

### Installation

- `git clone https://github.com/amirhh/menus.git`
- `cd menus`
- `npm install`
- `npm start` starts the local development server at http://localhost:8080
